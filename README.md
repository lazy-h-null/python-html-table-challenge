# Freshco Data to HTML Table (Daisy UI) Project

This project automates the creation of a styled HTML product table by fetching data from the Freshco API and using Python to generate a dynamic web page.

## 🎯 Assignment Goal
The goal is to fetch product data (Name, Price, Image) and render it into a structured HTML table using DaisyUI components and Python string manipulation.

## ✅ Assignment Checklist
- [x] 1. Use a `template.html` file with an empty `<body>` and DaisyUI/Tailwind links.
- [x] 2. Fetch live product data from the Freshco API.
- [x] 3. Use a Python `for` loop to iterate through the data list.
- [x] 4. Map and display `name`, `price`, and `image_url` for every item.
- [x] 5. Display images in `<img>` tags with a fixed size of 50x50 pixels.
- [x] 6. Select and implement a specific DaisyUI Table component.
- [x] 7. Add an `<h1>` title tag at the top of the table.
- [x] 8. Add two square buttons (Red: Delete / Green: Edit) in a new column.
- [x] 9. Test the layout manually with 1-2 rows before full rendering.
- [x] 10. Generate a full HTML string for the entire product list.
- [x] 11. Insert the table string into the body of the template file using `replace()`.
- [x] 12. Handle file reading (`template.html`) and writing (`index.html`) properly.
- [x] 13. Use additional DaisyUI elements (Stats/Product Count box) as a bonus.
- [x] 14. Share the website via GitHub Pages.
